You need to have dotnet installed on your system.

1.)
```
cd Creation
dotnet run
```

2.)
After it generates variables copy and paste them into the `Lyolan.cs` file
within Project.

3.)
Then run `dotnet build` to generate the DLL. (CHANGE THE VARIABLES BEFORE THIS STEP)

    HOW 2 SERVE:
This challenge you serve the DLL in a zip archive.

    SOLUTION:
Reverse Engineer the DLL to discover AES encryption and off-site pulling for
the KEY and IV, after reviewing the GetKeyAndIV() function and investigating
the URL it pulls from you can view the URL and get the flag

NO REMOTE CONNECTION REQUIRED
